# PaiMAN — коротко про гру та навчання

**PaiMAN** — це гра на JavaScript, у яку не можна грати вручну. Персонажем керує нейромережа, яка навчається за допомогою генетичного алгоритму.

## Як працює нейромережа

- У персонажа є сенсори, які визначають, чи є поруч стіна.
- Значення сенсорів подаються на вхід нейромережі.
- На основі цих даних і ваг мережі персонаж вирішує, куди рухатися (вихідний шар).

## Навчання

- На початку ваги нейромережі випадкові.
- Ви можете задати ваги навченої мережі у змінній `BestW` (рядок 111).
- Кожне покоління (100 циклів) обирається персонаж з найбільшою кількістю очок.
- На основі його ваг створюється наступне покоління.
- Для кожного нового персонажа 10% ваг випадково змінюються (мутація).

## make_dataset.html

- Створює дані для експорту в TensorFlow.
- Увімкніть ручне керування, щоб дані базувалися на ваших діях.
- Кожен рядок — це один хід: спочатку значення сенсорів, остання цифра — ваше рішення (напрямок руху).
